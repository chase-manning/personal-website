Other Cool Stuff

APIs

APIs are the language of the internet; they are how your Website and Mobile App get their data to display to the user. If you want to sound super fancy, you should totally get one of these! I like using C# Web APIs and Node.js.

Databases

Databases are where you store all your data for your Website or Mobile App. If you have some data to store, then you will probably need one of these! I like using MS SQL Server, MySQL and (if I'm felling hip) some NoSQL.

Dev Ops & CI/CD

Dev Ops is adding lots of awesome automation to make your development process faster and more stable. I like using Azure Pipelines, GitLab Runners, GitHub Workflows and Jenkins.
